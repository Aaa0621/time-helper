## 📌功能

- [x] 登录
  - [x] 密码登录
  - [x] 验证码登录
  - [x] Token登录
- [x] 打卡
  - [x] 使用最后一次打卡数据
  - [x] 随机体温范围
  - [x] 随机定位范围
  - [ ] 自动填写表单
  - [ ] 启动时间配置（[手动调节](#运行时间配置)）
- [x] 消息通知
  - [x] Server酱微信通知
  - [x] QMSG QQ通知
  - [x] 自定义推送服务
  - [x] 控制台输出
- [x] 命令行工具
  - [x] 打卡：`main`
  - [x] 获取打卡ID：`appId`
  - [x] 消息推送测试：`send`
  - [x] 随机位置生成测试：`geo`
  - [x] 设置向导：`setup`
  - [x] 获取令牌：`token`
